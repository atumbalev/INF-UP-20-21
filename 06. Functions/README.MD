### Задача 1

Да се напише програма, която въвежда цяло положително число n и печата правоъгълен триъгълник от звездички с размер n. \
Пример: \
Вход: 1 \
Изход: `*` \

Вход: 3 \
Изход: \
```
*
* *
* * *
```
